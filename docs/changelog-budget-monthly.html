Auto-copy behaviour

1
User navigates to a new month
Budget page month selector updates year/month state.
2
Fetch budgets for that month
fetchBudgets(userId, year, month) — returns empty if month is new.
3
Find most recent prior month
fetchMostRecentBudgets(userId, year, month) — scans all user's budget records, returns the latest month that predates the target.
4
Write copied rows to the DB
One upsertBudget per category — amounts carry over, year/month update to the new target.
5
Re-fetch and display
Inputs show the copied amounts. User edits save on blur — same behaviour as before.
